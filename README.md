# Movie Ticket Calculator

## Description

A beginner Python project that simulates a movie ticket booking system.
The program checks ticket eligibility, applies discounts, calculates extra charges, and determines the final ticket price based on user conditions.

## Features

* Age-based ticket eligibility checking
* Evening show restrictions
* Membership discount system
* Weekend and evening extra charges
* Different service charges for seat types
* Final ticket price calculation

## Technologies

* Python
* Conditional statements
* Boolean logic
* Variables and operators

## What I Learned

Through this project, I learned:

* How to use conditional statements (`if`, `elif`, `else`)
* How logical operators (`and`, `or`) work together
* How to calculate values dynamically
* How to structure a simple Python program
* How to simulate real-world booking logic using Python
